# 🚀 AASTHA INTELLIGENCE OS

A futuristic AI-inspired developer portfolio platform built with modern web technologies.

This project represents my personal developer identity as a **Full Stack Developer** and showcases my projects, skills, experience, and technical journey through an interactive operating system style interface.

---

## ✨ Features

* 🖥️ Custom OS-style boot sequence
* ⚡ Smooth animations and transitions
* 🌐 Interactive 3D core visualization
* 📂 Dynamic project showcase
* 🧠 Skill matrix dashboard
* 📄 Resume integration
* 🔗 GitHub & LinkedIn integration
* 📱 Fully responsive design

## 🚀 Run Locally

Clone the repository:

```bash
git clone https://github.com/aastha77/aastha-intelligence-os.git
```

Go into project folder:

```bash
cd aastha-intelligence-os
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
